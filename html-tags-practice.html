Headings

Headings h1 to h6 identify blocks and section. They also differenciate sizing of text. Every page must always have and only have an h1.

Anchors (i.e. links)

Anchor tags allow us to anchor a url into the webpage. Add 'target="_blank"'' to open link in a new tab.

App Academy

See MDN for more details on target=_blank.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to Jump to the top of the page. Just need to remember to type '#' before the id.

Text Emphasis

Italic

The `em` element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the `strong` element to emphasize text. This shows up as bold text in most browsers.
